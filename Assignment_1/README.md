# NLP Assignment 1

This repository contains the materials for **Natural Language Processing – Assignment 1**.

## 📂 Structure

- `Assignment_1.ipynb` – Jupyter notebook with all code, experiments and explanations.

## 🔍 Overview

The notebook explores core NLP concepts such as:

1. **Text preprocessing** (tokenization, normalization, stop‑word removal)
2. **Feature extraction** (bag-of-words, TF-IDF)
3. **Basic NLP tasks** (e.g. sentiment analysis, language modeling)  
4. Visualizations and brief discussion of the results.
